> ## [Plugins]
> **下面是我所使用的插件**
    >> plugin | 功能 
    >> |:----:|:----:|:----:|
    >> C/C++ | 提供C++的基本env
    >> C/C++ intellisense  | C++自动补全
    >> Code Runner | 安装后会在右上角出现'播放'按键,可以运行基本的C++小程序
    >> python | 链接系统中的python环境
    >> ROS | 在vscode中提供基本的ROS命令
    >> Txt Sytax | 提供txt的文本格式
    >> XML Format | 格式化XML文件工具
    >> vscode-icons | 美化文件在vscode中的图标
